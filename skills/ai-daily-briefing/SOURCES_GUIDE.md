## ⚠️ 有限制/需特殊处理 (3个源)

### 1. OpenAI News (openai.com)

**限制类型**: HTTP 403 Forbidden

**问题描述**: 
- RSS源被限制访问
- 可能需要特殊请求头或Cookie

**建议解决方案**:
1. 改用网页抓取替代RSS
2. 使用Playwright模拟浏览器访问
3. 寻找替代源（如OpenAI官方博客其他入口）

**手动获取方式**:
- 网址: https://openai.com/news
- 查看最新发布

### 2. 工信部 (miit.gov.cn)

**限制类型**: HTTP 403 Forbidden

**问题描述**: 
- 网站有反爬虫机制
- 直接请求返回403错误

**建议解决方案**:
1. 使用Playwright完整模拟浏览器
2. 添加完整请求头（Accept、Accept-Language、Referer等）
3. 使用代理IP轮换

**手动获取方式**:
- 网址: https://www.miit.gov.cn/
- 关注"政策文件"栏目
- 重点关注"人工智能"、"数字经济"相关政策

### 3. 湖北省政府 (hubei.gov.cn)

**限制类型**: HTTP 412 Precondition Failed

**问题描述**:
- 网站有预检请求限制
- 可能需要特定的请求头

**建议解决方案**:
1. 使用Playwright有头模式（非headless）
2. 确保执行页面JavaScript
3. 添加适当的延迟等待（10秒以上）

**手动获取方式**:
- 网址: http://www.hubei.gov.cn/
- 关注"政府信息公开"-"政策文件"
- 筛选省级AI产业政策
